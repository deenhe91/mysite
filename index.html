Hi, I'm a developer and data scientist.

A bit about me.

My passion for data and coding starting during my Master's Research, where I learnt to analyse and visualise large genomic datasets using R. Hooked, I went off to New York City to develop my skills in a data science bootcamp run by Metis. This is where my love for Python started and I learnt to use machine learning tools and build models. I got a job as a lead data scientist and become well-practiced in building robust data pipelines.

Though I enjoy playing with data, I missed the creativity and visual design aspect as my work was largely backend. I decided to take action and focus on learning more front-end skills. This website is the first result of that learning process. My favourite projects contain both visual and data aspects. I love communicating ideas to a wider audience and making beautiful frontends.

Hand-Coded HTML & CSS
